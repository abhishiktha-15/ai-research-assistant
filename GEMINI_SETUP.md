## Get Your Gemini API Key

1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Click "Create API Key"
3. Copy your API key

## Configure the Project

### Step 1: Navigate to Project
```powershell
cd E:\PROJECTS\RAG2
```

### Step 2: Create Environment File
```powershell
copy .env.example .env
```

### Step 3: Edit `.env` File

Open `.env` and add your Gemini API key:

```env
# API Keys
GEMINI_API_KEY=AIzaSy...your_actual_key_here

# Endee Configuration (default is fine)
ENDEE_HOST=localhost
ENDEE_PORT=8000

# Embedding Model (default: free Sentence Transformers)
EMBEDDING_MODEL=sentence-transformers
# Or use Gemini for embeddings too:
# EMBEDDING_MODEL=gemini

# LLM Model (Gemini is now default!)
LLM_PROVIDER=gemini
LLM_MODEL=gemini-1.5-flash

# RAG Settings (defaults are good)
CHUNK_SIZE=500
CHUNK_OVERLAP=50
TOP_K=5
SIMILARITY_THRESHOLD=0.7
```

## Optional: Start Endee Vector Database

If you have Docker:
```powershell
docker-compose up -d
```

Check it's running:
```powershell
docker ps
```

> **Note**: If you skip this, the system will use in-memory storage (works fine, but data doesn't persist).

## Add Your Research Papers

## Troubleshooting

**"Gemini API Error"**: Check your API key in `.env`

**"No PDFs found"**: Add PDFs to `data/papers/`

**"Cannot connect to Endee"**: Either:
- Start Docker: `docker-compose up -d`
- Or ignore (in-memory fallback works)
